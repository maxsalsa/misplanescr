## 3. ESTÁNDARES DE INGENIERÍA (STACK)
- **Frontend:** Next.js 15 (App Router) + TailwindCSS + DaisyUI.
- **Estética:** "Corporate Solemn" (Slate-900 / Blue-700). Cero "juguetes visuales".
- **Backend:** Server Actions + Prisma ORM.
- **Database:** Neon (PostgreSQL).
- **Seguridad:** Protocolo "Zero Trust". Un docente SOLO ve sus propios datos.
- **UX:** Prohibido `alert()`. Uso obligatorio de `sonner` (Toast) y Skeleton Loaders.

## 4. RIGOR PEDAGÓGICO (EL NÚCLEO)
- **Cronopedagogía:** El tiempo del plan debe coincidir con las horas reales del MEP.
- **Binomio Sagrado:** "El docente media... El estudiante construye...". Jamás usar lenguaje pasivo.
- **Inclusión Radical (Ley 7600/8899):** - La tabla `Student` manda.
  - Si hay banderas de Adecuación o Alta Dotación, el plan generado DEBE incluir apartados específicos.
